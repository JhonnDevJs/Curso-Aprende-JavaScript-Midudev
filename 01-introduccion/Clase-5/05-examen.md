# Examen Clase 5

## ¿Qué operador lógico se utiliza para comprobar si dos o más condiciones son verdaderas?

- OR (||)

- AND (&&)

Respuesta: AND (&&)

## ¿Qué operador lógico se utiliza para invertir el valor de una expresión booleana?

- OR (||)

- NOT (!)

Respuesta: NOT (!)

## ¿Qué valor devuelve la expresión false || true?

- false

- true

Respuesta: TRUE
